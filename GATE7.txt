SECTION H: THE MANDATORY PERSISTENCE TEST
===================================================================

This is the single test with real teeth. If this fails, the platform
is not fit for radio broadcasting. Period.

  1. Create a new station with a unique name
  2. Generate a song via AI Studio
  3. Add the song to a schedule
  4. Save the schedule
  5. Navigate away to a different page
  6. Navigate back to the station
  7. Verify the station, song, and schedule are intact
  8. Hard refresh the page (F5)
  9. Verify everything is STILL intact

  [ ] PASS — station and schedule survive navigation
  [ ] PASS — station and schedule survive hard refresh

  IF EITHER FAILS: StruthRadio is NOT a functional radio platform.
  Do not continue feature work. Fix persistence first.
